Project

โปรเจค การสร้างแบบจำลองและ โค้ดสำหรับอุปกรณ์ระบุตำแหน่ง ด้วยเสียงผ่าน LabVIEW และ MATLAB

เป็นโปรเจคจบ จากมหาวิทยาลัยศิลปากร ใช้โปรแกรม LabVIEW และ MATLAB ในการสร้างโค้ดสำหรับจับทิศทางเสียงของผู้พูดโดย ไม่ให้กล้องหมุนไปหาผู้พูดแต่เป็นการซูมเข้าไปที่ใบหน้าผู้พูดโดยใช้การคำนวนด้วยเสียง หลังจากนั้นจะจับหน้าผู้พูดไว้จนกว่าผู้พูดจะพูดจบ แล้วจับเสียงคนต่อไป

มินิ โปรเจค นาฬิกาจับเวลาด้วย MPLAB IDE และ Arduino

เป็นการใช้โปรแกรม ISIS ในการสร้างบอร์ด เพื่อใช้ในการประกอบนาฬิกาจับเวลาและ ใช้โปรแกรม Ardiuno IDE และ MPLAB IDE ในการเขียนโค้ด ออกมา โดยสามารถจับเวลาได้ นานสุด 12 ชั่วโมง

มินิ โปรเจค สำหรับเก็บข้อมูล ใน DEV C++

ใช้โปรแกรม DEV C++ เพื่อสร้าง Table สำหรับเก็บข้อมูลของบุคคล โดยทำให้คนแต่ละคนมีการสร้างไอดีขึ้นมาไม่เหมือนกัน แล้วจัดเก็บข้อมูลชื่อไว้ใน Server ของมหาลัย
